Git研修

個人メモ

クローン先のディレクトリに入る＞＞cd Desktop/（など）

ディレクトリに入った状態でクローン＞＞git clone ○○○○（SSH） △△△△（保存先）

ディレクトリの中にデータが入っているか確認

自分の作業用ブランチを作る＞＞git checkout -b ishii(任意の名前）

中のデータをいじる＞＞vi ○○○○（ファイル名）※もしくはVSコードなど

編集したデータをstaging areaにあげる＞＞git add （ファイル名）

staging areaにあげたデータを確認＞＞git status

staging areaにあげたデータをコミットする＞＞git commit -m 'コメント'

コミットしたデータを確認＞＞git status

メインのブランチから最新データを反映させる＞＞git pull origin メインのブランチ名

自分が編集したデータをメインのブランチにプッシュする＞＞git push origin 自分の作業ブランチ名
